## 本書の概要

Webサイトで公開できる対話的・探索的（読み手が自由に動かせる）可視化をPythonで実践します．データ解析に便利なPlotly，アプリ化のためのユーザインタフェースを作成できるDash，ネットワーク図に強いDash Cytoscapeを具体的に解説しています。

目次    
**0章** **はじめに**    
**1章** **Plotly Express**     
**2章** **plotly.py入門**     
**3章** **plotly.pyの様々なグラフ**     
**4章** **plotly.py応用**    
**5章** **Dash入門**     
**6章** **Dashレイアウト**     
**7章** **Dashコールバック**     
**8章** **Dashの標準コンポーネント**     
**9章** **Dashの追加コンポーネント**     
**10章** **Dash Cytoscape入門**     
**11章** **Dash Cytoscape応用**
